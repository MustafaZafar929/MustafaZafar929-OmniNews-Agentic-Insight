# ✨ OmniNews Agent: Premium Feature Roadmap

To make this project a true "portfolio piece" that wows recruiters or instructors, we can move beyond simple summarization and into **Geopolitical Intelligence**. Here are four standout feature blocks:

---

### 1. 🌍 Geopolitical Risk & Impact Scoring
Instead of just summarizing, the agent predicts the **consequences** of the news.
- **The Feature:** A "Stability Index" (1-10) for every briefing.
- **Logic:** The Researcher agent is tasked with finding how this event affects:
    - Global Markets (Oil, Tech, etc.)
    - Regional Security
    - Diplomatic Relations
- **Visual:** A color-coded "Risk Badge" on the frontend (Green = Stable, Red = High Tension).

### 2. 🕵️‍♂️ Cross-Media Bias Analysis
Demonstrate the power of the "Critic" agent by comparing perspectives.
- **The Feature:** A "Perspective Split" view.
- **Logic:** The agent specifically looks for a Western Source (e.g., CNN) and an Eastern/Regional Source (e.g., Al Jazeera or Xinhua) for the same event. 
- **Output:** A small table showing: *"How Source A framed it"* vs *"How Source B framed it."*

### 3. 🕸️ Entity Knowledge Graph (Visual)
Show the "Long-Term Memory" in action.
- **The Feature:** An interactive map of related players.
- **Logic:** As the agent researches, it extracts **Entities** (People, Countries, Organizations).
- **Visual:** A small graph on the dashboard showing connections (e.g., "Israel" --[Conflict]--> "Hezbollah" --[Support]--> "Iran").

### 4. 🎙️ AI "Podcast" Briefings (Multimodal)
Convert the written briefings into an audio experience.
- **The Feature:** A "Listen to Briefing" button.
- **Logic:** Use OpenAI or Google TTS to generate a high-quality audio summary. 
- **Standout Factor:** Mention in your report that the script was written by an agent to be "Radio Ready" (switching between a host and a field reporter voice).
